# Vaishnavi Subramanian Portfolio

This is a modern responsive static portfolio website built with React + Vite and styled with Tailwind CSS.

## Features

- Responsive design (mobile/tablet/desktop)
- Smooth scrolling navigation
- Reusable components
- Tailwind CSS for styling
- React Icons for iconography
- Static site, no backend required

## Setup

```bash
npm install
npm run dev
```

The site will be available at `http://localhost:5173` by default.

## Build

```bash
npm run build
npm run preview
```
